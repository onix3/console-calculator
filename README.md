# Console Calculator
A simple calculator with user interface in console

## Features
  - Mouse support
  - Input is valid
  - Basic operators only: `*` `/` `+` `-`

## Installation
Execute:
```
go get github.com/onix3/console-calculator
```
You will find the executable file `console-calculator` in `$GOPATH/bin/`
